\doxysubsection{Detailed Description}
The \doxylink{class_media_manager}{Media\+Manager} class manages multimedia objects and groups. 

The \doxylink{class_media_manager}{Media\+Manager} class provides methods to create, display, play, and remove multimedia objects, as well as create and remove groups. It uses a map to store multimedia objects and groups.
