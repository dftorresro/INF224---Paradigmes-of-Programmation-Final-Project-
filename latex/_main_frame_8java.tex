\doxysubsection{Detailed Description}
This file contains the \doxylink{class_main_frame}{Main\+Frame} class which is a part of the swing package. 

The swing package provides a set of GUI components for creating graphical user interfaces. The \doxylink{class_main_frame}{Main\+Frame} class is a top-\/level container that represents the main window of an application. It extends the JFrame class from the javax.\+swing package.
